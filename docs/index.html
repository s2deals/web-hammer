Web Hammer (Refactored)

A modern Layer 7 DoS stress-testing tool (for educational purposes)

⚠️ CRITICAL WARNING ⚠️

This tool is a Denial-of-Service (DoS) script. Using it against any website or server you do not own or have explicit permission to test is illegal and unethical. This code is provided for EDUCATIONAL PURPOSES ONLY. You are solely responsible for your actions.

About

This is a refactored version of the original "Hammer Dos Script". It has been updated to follow modern Python 3 best practices, improve efficiency, and enhance robustness. It uses two methods to generate traffic:

  • Direct Socket Connections: Sends raw HTTP GET requests directly to the target.
  • Proxy Bot Requests: Leverages third-party services to send requests on behalf of the script.

Quick Start

Run the script from your terminal. The only required argument is the target server.

python3 hammer.py -s example.com -p 80

For more options and details, see the full documentation on GitHub.
